# page-jisusaiche-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面各自承载不同的内容与用途，彼此之间无直接关联。仓库本身不针对任何特定网站或域名，仅作为静态页面的存储与分发场所。

## 目录说明

```
├── index.html          # 仓库首页，列出所有归档页面
├── pages/              # 存放各个独立 HTML 页面的文件夹
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
├── README.md           # 本文件
└── LICENSE             # 许可证文件
```

## 页面归档说明

- 每个页面均为独立 HTML 文件，可单独访问。
- 页面内容可能涵盖笔记、工具、演示、实验等不同类型。
- 部分页面可能引用 `assets/` 目录下的公共资源。
- 页面文件命名采用 `page-xxx.html` 格式，以便于索引。

## 维护说明

- 本仓库由个人维护，不定期更新。
- 新增页面时，会同步更新 `index.html` 中的链接列表。
- 若发现页面错误或链接失效，欢迎提交 Issue。
- 不接受针对具体网站或商业用途的页面添加请求。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 即可浏览所有页面。
3. 也可直接访问 `pages/` 目录下的单个 HTML 文件。
